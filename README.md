# 🔢 Count App

แอปนับตัวเลขแบบง่ายๆ สร้างด้วย HTML, CSS และ JavaScript ล้วนๆ ไม่มี framework ใดๆ

## ✨ Features

- **เพิ่ม / ลด** ค่าทีละ 5
- **ตัวเลขติดลบไม่ได้** — ค่าต่ำสุดคือ 0
- **สีเปลี่ยนตามค่า** — เขียว (บวก), ดำ (ศูนย์)
- **จำค่าได้** — บันทึกลง `localStorage` ค่าไม่หายหลัง refresh
- **ปุ่ม Reset** — รีเซ็ตกลับเป็น 0 ได้ทันที
- **Responsive** — ใช้งานได้ทั้งมือถือและคอมพิวเตอร์

## 🗂️ โครงสร้างไฟล์

```
count-app/
├── index.html   # โครงสร้างหน้าเว็บ
├── style.css    # ดีไซน์และ layout
└── app.js       # logic การทำงานทั้งหมด
```

## 🚀 วิธีใช้งาน

ไม่ต้องติดตั้งอะไรเพิ่มเติม เปิดไฟล์ได้เลย

```bash
# Clone repo
git clone https://github.com/Youngmaidainon/Count-app

# เปิดไฟล์ในเบราว์เซอร์
open index.html
```

หรือจะลากไฟล์ `index.html` วางในเบราว์เซอร์โดยตรงก็ได้

## 🖼️ หน้าตาแอป

| ปุ่ม | การทำงาน |
|------|----------|
| `+` | เพิ่มค่าขึ้นทีละ 5 |
| `-` | ลดค่าลงทีละ 5 (ต่ำสุด 0) |
| `Reset` | รีเซ็ตค่ากลับเป็น 0 |

## 🛠️ เทคโนโลยีที่ใช้

- HTML5
- CSS3 (Flexbox, CSS transitions)
- JavaScript (localStorage API)
